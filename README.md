 # Python Projects & Automation Tools

A collection of Python projects developed to practice software engineering logic, system administration, and computer science fundamentals.

## 🛠️ Projects Included
* **`sysadmin_cli.py`**: Command Line Interface (CLI) tool for Linux system administration and task automation.
* **`length_converter.py`**: Unit conversion utility handling numeric data inputs and calculations.
* **`Calculator.py`**: Command line calculator for basic mathematical operations.

## 🎯 Main Goal
To continuously build practical tools and strengthen core skills in Python development, Linux integration, and backend logic.
